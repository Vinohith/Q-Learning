# Q-Learning
Q-Learning implementation on the Mountain Car environment from OpenAI Gym.

<br>
<br>

## Requirement

- Python 3.7.3 
- gym 0.14.0
- numpy 1.17.0

## Setup
### Install dependencies
```
pip install gym
pip install numpy
```
### Run the code
```
git clone https://github.com/Vinohith/Q-Learning.git
cd Q-Learning
python q_learning.py
```
